시작에 앞서... 
==============
동네API는 RESTful API를 제공하며 인증 방식으로 OAuth 1.0을 사용합니다. OAuth는 처음 API를 사용하는데 있어 진입 장벽이 될 수 있지만, 동네의 보호된 자원을 안전하게 3rd-party 애플리케이션에 공개할 수 있는 방식입니다. 

동네API는 트위터의 RESTful API를 참고로 구현하였으므로 사용법이 유사합니다. 트위터 API를 사용한 경험이 있다면 동네API도 어렵지 않게 사용할 수 있습니다.

Consumer 등록
=============
동네API를 통해 동네의 자원(protected resource)을 사용하려면 Consumer 등록을 해야합니다.
