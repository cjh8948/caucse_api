=======
동네API
=======

동네API는 동문 네트워크 서비스('동네'로 알려진)의 Open API 서비스입니다. **동문 여러분에 한해** 10여년간 축적된 동문 네트워크의 자료를 Open API를 통해 사용할 수 있도록 공개합니다.

동네API를 이용해서 동네M과 같은 3rd-party 애플리케이션을 만들 수 있으며 트위터, 페이스북과의 매쉬업도 가능해집니다. 아이폰이나 안드로이드용 앱과 같은 네이티브 애플리케이션 구현도 가능합니다.

동네API는 트위터의 RESTful API를 참고로 구현하였으므로 사용법이 유사합니다. 트위터 API를 사용한 경험이 있다면 동네API도 어렵지 않게 사용할 수 있습니다.
